## reguloj

Vortordo en o-frazoj estas:
determiners (hin/den/moy/nil) -- possessive adj (misu, etc.) -- number --
adjective -- (pro)noun

1. difiniloj (hin/den/moy/nil = ĉi tiu / tiu / ĉiu / neniu) 
2. poseda adjektivo (misu, yusu = mia, via, ... ) 
3. nombro
4. adjektivo 
5. pronomo / substantivo
